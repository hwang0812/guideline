DENTRION GUIDE LINE

파트너 관리

이 화면은 파트너의 정보를 확인 및 수정을 할 수 있으며, 파트너 추가가 가능합니다.


1-1.기능 설명

1-1-1.목록

파트너 검색 및 목록을 확인할 수 있습니다.


1-1-2.파트너 기본 정보

등록된 파트너의 정보를 확인할 수 있습니다.

1.파트너 : 등록된 파트너명입니다.

2.대표 : 등록된 파트너의 대표자 이름입니다.

3.회사 주소 : 파트너의 주소입니다.

4.회사 전화번호 : 파트너의 대표 연락처입니다.

5.사업자등록번호 : 파트너의 사업자등록번호입니다.

6.지역 정보 : 국내/해외 지역이 표시됩니다.


1-1-3.파트너 상세 정보

등록된 파트너의 세부 사항을 확인 및 수정이 가능합니다.

1.거래명세서 기간 : 파트너의 거래명세서 기간을 설정하여 해당 기간에 포함된 의뢰가 자동으로 쌓이고 거래명세서가 생성됩니다.

2.기공 할인율 : 할인율이 있는 경우 적용하여 거래명세서 생성 시 할인율이 자동으로 반영됩니다.

3.공급자 : 파트너와 연결된 공급자를 확인 및 수정이 가능합니다.

4.단축어 : 의뢰서 작성 또는 파트너 검색 시 빠르게 검색할 수 있도록 설정한 약어입니다.


1-1-4.파트너 추가

신규 파트너를 추가 할 수 있습니다.


1.병원명(필수입력항목) : 추가할 병원명을 입력합니다.


2.주소 : 병원 주소를 입력합니다.


3.대표자 : 병원 대표자명을 입력합니다.


4.병원 전화번호 : 병원 대표 연락처를 입력합니다.


5.사업자등록번호 : 병원 사업자등록번호를 입력합니다.


6.사업자등록증 : 병원 사업자등록증을 JPG,JPEG,PDF,PNG 파일로 업로드합니다.


7.지역 설정 : 국내/해외 지역 설정을 합니다.


8.추가하기 : 필수 입력값을 입력하면 버튼이 활성화됩니다.
